# RSI-HUB-TDD
Projeto de testes com TDD

O Projeto tem 4 Pacates onde cada um tem una funçao ao teste com 3 cenarios com uma teste positivo e negativo.

Pacote- Page factory
É responsavel pela mapeamento e ação do testes.

Pacote - Utility 
Contem a classes que configura o excel e o print de tela.

Pacote- Positivo
tem tres cenarios com testes positivos.

Pacote- Negativo 
Com os 3 cenarios negativos.
